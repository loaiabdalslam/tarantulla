SUÍTE TARANTULLA

O Tarantulla é um projeto Open Source com módulos especialistas para extração e estruturação de dados da Web.

< Faça você mesmo />

  • Tarantulla-Facebook é o modulo responsável por crawler e scraping de informações públicas no Facebook dos publishers escolhidos, usando a Graph API do Facebook. O tarantulla-facebook é desenvolvido em Python 3.x e utiliza o Pentaho Data Integration - PDI para a estruturação dos dados*.

    Acessar projeto

  • Tarantulla-Twitter é o módulo responsável por crawler e scraping de informações públicas no Twitter dos publishers escolhidos, usando a Timeline API do Facebook. O tarantulla-twitter é desenvolvido em Python 3.x e utiliza o Pentaho Data Integration - PDI para a estruturação dos dados*.

    Acessar projeto

  • Tarantulla-Youtube é o módulo responsável por crawler e scraping de links do YouTube por tópico de interesse. O tarantulla-youtube é desenvolvido em Python 3.x e utiliza o Pentaho Data Integration - PDI para a estruturação dos dados*.

    Acessar projeto

  • O Tarantulla-Web é o módulo responsável por crawler e scraping de páginas web. O tarantulla-web é desenvolvido em Python 3.x e utiliza o Pentaho Data Integration - PDI para a estruturação dos dados*.

    Em breve

  • A utilização do Pentaho Data Integration é opcional, você pode exportar os dados usando JSON.
